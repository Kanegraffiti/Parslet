What is Parslet?

Parslet is a minimal DAG engine that lets you connect small Python functions (tasks) and run them in the right order, on devices with limited power, memory, or connectivity.

Why Parslet?

Offline-first

Design workflows that survive flaky or absent connections.

Battery-aware scheduling

Balance performance with power budgets on phones and edge boards.

Android/Termux-ready

Ship DAGs directly to mobile devices for on-the-go processing.

Quickstart

pip install --upgrade pip
pip install parslet
pkg update -y && pkg install -y python graphviz
python -m venv .venv && source .venv/bin/activate

pip install parslet

Hello DAG

from parslet import parslet_task

@parslet_task
def hello():
    print("hi")

if __name__ == "__main__":
    from parslet import run
    run([hello])

Run it:

parslet run my_flow.py
Troubleshooting

Ensure Graphviz is installed and available on PATH. On Termux, grant storage permissions if exporting images.
