Quickstart

# Install from PyPI
pip install parslet
from parslet import parslet_task, ParsletFuture, DAG, DAGRunner
from typing import List

@parslet_task
def hello(name: str) -> str:
    return f"Hello, {name}!"

@parslet_task
def loud(text: str) -> str:
    return text.upper()

def main() -> List[ParsletFuture]:
    a = hello("Parslet")
    b = loud(a)
    return [b]

if __name__ == "__main__":
    dag = DAG()
    dag.add_tasks(*main())
    DAGRunner(dag).run()

Run it:

parslet run my_workflow.py

Export a DAG image (requires graphviz):

parslet run my_workflow.py --export-png dag.png

Interop (experimental)

Convert simple scripts to move between ecosystems.

# Parsl → Parslet
parslet convert --from-parsl app.py --to-parslet out.py

# Dask → Parslet
parslet convert --from-dask dask.py --to-parslet out.py
