What is Parslet?

Parslet is a pocket-sized workflow engine for Python. You write small functions (tasks), Parslet links them as a DAG and runs them efficiently—scaling down on low battery, working happily offline, and fitting neatly on Android via Termux or tiny edge devices.

  • Define tasks with a decorator, return a final list of futures—that’s your workflow.
  • Adaptive, battery-aware runner with sensible defaults and a simple CLI.
  • Experimental import/export to Parsl and Dask for portability.

Offline-first

Run workflows without connectivity. Great for field work and clinics.

Battery-aware

Adaptive scheduling that scales down when power is scarce.

Tiny & Pythonic

Just write functions. Parslet handles dependencies as a DAG.

Practical Use-Cases

Battery-Aware Data Pipeline

Check battery → choose quick or full analysis → save results. Great for phones in the field.

RAD by Parslet (Triage)

Ingest → preprocess → two light models → agree=auto-note, disagree=human review.

Edge MCU Sensor Processing

Batch read → clean → feature extract → store/export; resilient on low RAM.

Text Cleaner

Load text → normalize → dedupe → save, useful for SMS/USSD exports.

Image Filter

Load image → grayscale → blur → save; ideal for lightweight image ops.

Parsl/Dask Interop (exp.)

Convert simple scripts to move between ecosystems with a CLI bridge.

Quickstart

Docs: See the full picture on Read the Docs.

pip install parslet
parslet run my_workflow.py

Export a DAG image (requires graphviz):

parslet run my_workflow.py --export-png dag.png

Interop (experimental)

parslet convert --from-parsl app.py --to-parslet out.py
parslet convert --from-dask dask.py --to-parslet out.py

Android / Termux

pkg install python
pip install parslet
termux-setup-storage
parslet run examples/battery_aware_demo.py
